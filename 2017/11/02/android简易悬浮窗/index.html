android简易悬浮窗

发表于 2017-11-02 | 分类于 android | 0 comments | 阅读次数:

一.前言

android中经常会有一些桌面的小部件，比如360，或者一些录屏软件，这些小部件就叫做悬浮窗。

二. 主要过程

  • 添加权限
  • 添加LayoutParams,决定悬浮窗的位置,展现形式
  • floatView Constructor
  • floatView create
  • add view to window
  • move listener

flag

FLAG_ALT_FOCUSABLE_IM => 获取焦点，且其他app无法获取焦点

type

WindowManager.LayoutParams.TYPE_TOAST 可以显示一个悬浮窗且不需要权限，但是在点击home后没法在桌面显示

params.gravity

Gravity.TOP | Gravity.LEFT 初次加载的位置，会决定OnTouchEvent中event.getRawX || event.getRawY 的值

updateViewLayout

悬浮窗跟随手指移动，windowManager.updateViewLayout(overlayView, params);
