Adobe Air在android上播放flash

发表于 2017-11-01 | 分类于 Adobe Android | 0 comments | 阅读次数:

前言

 Adobe已经放弃移动端对flash的支持，因此想要在新版本的android，其实已经是市面上大多数的android设备上播放flash需要其他的方法。

两种方式

  • Tbs腾讯浏览服务
  • Adobe Air开发

Tbs腾讯浏览服务

tbs腾讯浏览服务可以理解为一个加载了X5内核的webview,支持html5,打开各种文件，视频等，但还是不直接支持flash，要支持adobe插件。下载地址 https://adobe-flash-player.en.softonic.com/android/download 。
正常使用的话需要以下两种配置就可以在qq浏览器中正常播放flash了

  1. qq浏览器
  2. adobe插件

作为开发者接入的话，需要以下三种

  1. tbs sdk 需要老版本的
  2. X5内核，应用可以与qq，微信，qq浏览器等工序X5内核
  3. adobe插件

比较坑爹的是只有老版本的tbs sdk才能播放flash，官网最新的sdk无法播放flash，而且支持flash的script2.0

  1. 下载TBSDemo
  2. 下载插件

Adobe Air开发

1. download Adobe Air SDK

http://www.adobe.com/devnet/air/air-sdk-download.html

mac上下载完成后安装后，因为是安装在 /Volumes目录下 将Air sdk内的所有文件复制到 /Users/yourname/empty-dirctory 目录下

2. Path environment variables

https://help.adobe.com/en_US/air/build/WSfffb011ac560372f-71994050128cca87097-8000.html#WS2d929364fa0b8137-78b25bdf12a498897d1-8000

设置环境变量完成检查是否成功

3. build your first hello world app

https://help.adobe.com/en_US/air/build/WS901d38e593cd1bac25d3d8c712b2d86751e-8000.html#WS901d38e593cd1bac25d3d8c712b2d86751e-7ffb

problems,遇到的问题

1. amxmlc HelloWorld.as

Q: initial content not found
A: https://stackoverflow.com/questions/13475655/how-do-i-fix-a-initial-content-not-found-error

2. adl HelloWorld-app.xml

Q: error while loading initial content
A: https://stackoverflow.com/questions/9537076/chart-of-swf-versions-to-flash-versions/10900792#10900792

这个是因为swf的version和编译环境air的version需要map
,这个是查看map的地址：http://www.adobe.com/devnet/articles/flashplayer-air-feature-list.html

查看swf version
修改air version
3. 生成签名文件
4. 打包生成apk

Q: adt and a password may be incorrect
A: adt -package -target apk -storetype pkcs12 -keystore sampleCert.pfx HelloWorld.apk HelloWorld-app.xml HelloWorld.swf 这个是因为sampleCert.pfx 和 sampleCert.p12 没有对应

5. Install the AIR runtime 下载手机上运行air的环境
adt -installRuntime -platform android -platformsdk path-to-android-sdk
6. Install the Helloworld app 安装helloworld app
adt -installApp -platform android -platformsdk path-to-android-sdk -package path-to-app/appname.apk
7. Uninstall the Helloworld app 卸载helloworld app
adt -uninstallApp -platform android -appid samples.android.HelloWorld
8. add android descriptor 添加AndroidManiFest标签

Q: error 100: Descriptor cannot be parsed
